# Poker Decision Helper

A static website for GitHub Pages.

## File Structure

- `index.html`: website structure
- `style.css`: visual design
- `script.js`: poker action recommendation logic

## How to Use

1. Upload `index.html`, `style.css`, `script.js`, and `README.md` to your GitHub repository.
2. Go to **Settings → Pages**.
3. Set the source to **Deploy from a branch**.
4. Select **main** and **/root**.
5. Save and open the published GitHub Pages link.
